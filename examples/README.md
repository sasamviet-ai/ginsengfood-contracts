# examples

Thư mục này chứa request, response, event và error mẫu để hỗ trợ dev, QA, contract test và debugging.

Examples không phải source-of-truth nghiệp vụ; chúng chỉ minh họa contract đã được định nghĩa ở `openapi/`, `asyncapi/`, `schemas/`, `events/` hoặc `error-codes/`. Không đặt dữ liệu khách hàng thật, secret, production dump hoặc business logic vào đây.

Example nên đi kèm version contract mà nó minh họa, ví dụ `order-confirmed-v1.example.json`.
