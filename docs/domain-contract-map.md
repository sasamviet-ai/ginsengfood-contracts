# Phase 0 Domain Contract Map

This file maps expected contract domains to source documents. It is a planning artifact only; it does not define real schemas, enums, APIs, events, state machines, or error codes.

Every future contract output must be versioned `v1` and must cite the source document sections used to derive it.
