# Error Code Standard

Tài liệu này định hướng cách đặt error code để API không trả lỗi ngẫu nhiên theo từng service.

Error code cần ổn định, có namespace rõ, có mô tả, HTTP status tương ứng nếu áp dụng, và có ví dụ trong `examples/errors/` trước khi production dùng.

Danh sách error code chuẩn thuộc `error-codes/`, không đặt danh sách đầy đủ trong tài liệu này.
