# Release Checklist

Checklist trước khi release contract:

- API/event/schema đã versioned.
- Breaking change đã được provider và consumer review.
- Contract quan trọng đã có schema.
- Contract dùng production đã có example.
- Changelog đã được cập nhật.
- Migration note đã có nếu thay đổi phá vỡ compatibility.

Không đặt script release hoặc generated artifact vào tài liệu này.
