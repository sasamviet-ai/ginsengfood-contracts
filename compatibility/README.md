# compatibility

Thư mục này chứa chính sách breaking change, deprecation, version matrix và consumer compatibility.

Người dùng chính là provider owner, consumer owner, release manager, QA và reviewer contract. Không đặt implementation workaround, migration SQL, deployment config hoặc generated artifact vào đây.

Ví dụ file sau này: `version-matrix.md`, `deprecation-policy.md`, `consumer-compatibility.md`, `breaking-change-review.md`. Mọi breaking change phải có review và migration note.
