# contract-tests/postman

Thư mục này dành cho Postman collection hoặc mô tả test API trong tương lai.

Người dùng chính là QA, backend và CI nếu sau này có policy. Không đặt token production, environment secret, generated client hoặc test implementation chưa được review vào đây.

Ví dụ file sau này: `ops-core-api-v1.postman_collection.json`, `business-platform-api-v1.postman_collection.json`. Collection phải khớp version OpenAPI tương ứng.
