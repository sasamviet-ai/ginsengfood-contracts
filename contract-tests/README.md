# contract-tests

Thư mục này chứa định hướng và tài sản phục vụ contract testing giữa `ginsengfood-ops-core` và `ginsengfood-business-platform`.

Hiện chưa tạo test thật. Không đặt application implementation, dependency package, generated SDK hoặc secret vào đây.

Các nhóm dự kiến gồm `postman/`, `pact/` và `fixtures/`. Contract tests phải tham chiếu version contract cụ thể để tránh test nhầm interface.
