# contract-tests/pact

Thư mục này dành cho Pact contract tests trong tương lai.

Người dùng chính là provider/consumer team khi cần kiểm chứng compatibility tự động. Không đặt application code, dependency install script, secret hoặc generated SDK vào đây.

Ví dụ file sau này: `ops-core-provider-v1.pact.json`, `business-platform-provider-v1.pact.json`. Pact file phải gắn với version API/event/schema cụ thể.
