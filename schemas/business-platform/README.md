# schemas/business-platform

Provider-level schema boundary index for `ginsengfood-business-platform`.

This folder does not define duplicate domain schemas. It maps business-platform provider responsibilities to the existing v1 domain schemas under `schemas/common`, `schemas/commerce`, `schemas/channel`, `schemas/ai`, `schemas/ads`, `schemas/live`, and `schemas/ivr`.

Allowed content:

- Provider schema indexes.
- Provider boundary notes.
- Compatibility notes for schema consumers.

Not allowed:

- Implementation code.
- Database schema or migration.
- Service logic.
- Business logic.
- Generated clients.
- Duplicated domain schema definitions.
