# generated

Thư mục này dành cho client/model generated nếu sau này repo quyết định commit generated files theo policy.

Mặc định chưa commit generated code thật. Không đặt generated code bắt buộc, build output tạm, dependency package, secret hoặc hand-written implementation vào đây.

Các thư mục `dotnet/`, `java/`, `typescript/` chuẩn bị cho .NET, Spring Boot và Next.js. Nếu sau này dùng, generated artifact phải ghi rõ source contract version và cách tái tạo.
